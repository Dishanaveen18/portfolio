# Disha Naveen - Personal Portfolio Website (HTML Only)

## Project Overview
This is a personal portfolio website built using **pure HTML only** (no CSS, no JavaScript) to showcase my education, technical skills, projects, and professional achievements. The website demonstrates fundamental HTML structure and semantic markup while serving as a digital resume.

## Objectives
* Create a functional portfolio using only HTML.
* Demonstrate mastery of HTML5 semantic elements.
* Showcase technical skills and projects effectively.
* Provide easy contact methods for potential employers.
